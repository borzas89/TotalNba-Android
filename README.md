# TotalNba-Android

Android application for my TotalNBA server app, app displays predictions for NBA games.

https://totalnba.herokuapp.com/api/prediction/all-prediction

Used technologies:
- Kotlin
- Dagger-Hilt
- Retrofit
- RxJava
- ViewModel
- Jetpack Navigation, Compose
- Databinding
- Room
